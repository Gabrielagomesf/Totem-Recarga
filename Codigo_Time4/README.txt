Ao abrir o projeto pelo vscode digite abra um terminal, e digite respectivamente:

npm install
npm run app

Após isso entre no vpn da puc

Após esses procedimentos será possível rodar o projeto normalmente

Caso dê um erro falando que o windows não pode rodar scripts ou algo assim
abra o PowerShell como administrador e digite:

Set-ExecutionPolicy Unrestricted
A + ENTER

Tente executar novamente.
